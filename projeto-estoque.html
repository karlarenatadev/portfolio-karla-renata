Machine Learning No-Code na AWS

AWS SageMaker Time Series ML No-Code Inventário

🚨 O Desafio

Otimizar a gestão de inventário de varejo para evitar rupturas de estoque ou excesso de produtos parados, utilizando modelos preditivos que considerem variáveis como preço e promoções.

💡 A Solução

Desenvolvi um modelo de Time Series Forecasting utilizando o Amazon SageMaker Canvas:

  • Tratamento de Dados: Uso do Data Flow para garantir que identificadores de produtos fossem tratados corretamente como texto.
  • Treinamento preditivo: Configuração do modelo para prever a quantidade de estoque para os próximos 5 dias.
  • Análise de Impacto: Identificação de quais fatores (como o preço) mais influenciavam a baixa do estoque.

🚀 Resultados

Obtenção de métricas sólidas de performance, com um Avg. wQL de 0.188, indicando alta precisão nas previsões. O modelo revelou que o preço tem um impacto de aproximadamente 25,44% na variação do estoque.
