AWS Certification Simulator

IA Generativa & Engenharia de Dados (Pipeline ETL)

Python Google Gemini Pydantic PWA

🚨 O Desafio

Garantir a qualidade de simulados para certificações AWS sem depender de questões estáticas ou desatualizadas. O maior desafio era evitar que a IA gerasse "alucinações" (respostas erradas ou serviços fora do escopo do exame).

💡 A Solução

Construí um ecossistema completo focado em Data Quality. O backend em Python utiliza a API do Google Gemini com um pipeline ETL autônomo que possui 4 camadas de validação rigorosa:

  • Schema Enforcement: Validação estrutural via Pydantic para garantir o formato JSON correto.
  • Semantic Validation: Um validador de negócio que impede serviços fora do escopo (ex: reprova Transit Gateway em exames de Cloud Practitioner).
  • Deduplication: Uso de SequenceMatcher para garantir que questões similares não sejam repetidas no banco.
  • Frontend Modular: Um PWA em Vanilla JS que consome esses dados e funciona offline.

🚀 Resultados

Geração de datasets limpos e validados para diversas certificações (CLF-C02, SAA-C03). A plataforma oferece gráficos Radar de desempenho e gamificação persistida no navegador.
