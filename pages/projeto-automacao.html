Automação de Cadastro (RPA)

Auditoria e input de dados automatizado em ERP

Python PyAutoGUI Pandas Automação

target O Desafio

O processo manual de cadastro de produtos consumia horas valiosas da equipa e estava sujeito a "erros de dedo" (typos), como preços incorretos ou códigos duplicados. Além disso, a repetição mecânica gerava fadiga nos colaboradores, reduzindo a produtividade em tarefas mais analíticas.

smart_toy A Solução

Desenvolvi um Bot (Robô) em Python que atua como um funcionário digital. Ele lê bases de dados, aplica regras de negócio e interage diretamente com a interface do sistema ERP simulando interações humanas.

  • Pandas: Utilizado para extrair dados de ficheiros Excel/CSV e realizar a higienização prévia (tratamento de células vazias e formatação).
  • PyAutoGUI: Controla o mouse e o teclado para navegar no sistema ERP de forma autônoma.
  • Lógica de Espera (Wait): O código é dinâmico e inteligente o suficiente para aguardar o tempo de carregamento do sistema entre uma tela e outra, evitando quebras na execução.

rocket_launch Resultados

O script tornou-se capaz de auditar e cadastrar centenas de produtos em poucos minutos com 100% de precisão. Este ganho de eficiência operacional libertou a equipa comercial para focar em tarefas de negociação e estratégia, eliminando completamente o gargalo do backoffice.
