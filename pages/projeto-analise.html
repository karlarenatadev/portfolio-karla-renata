Análise Exploratória de Vendas

Descobrindo padrões de lucro e comportamento do consumidor

Python Pandas Seaborn Storytelling

target O Problema de Negócio

Uma empresa de varejo precisava entender por que, apesar do alto volume de vendas em certas regiões, o lucro final estava estagnado. O meu objetivo foi identificar de forma cirúrgica quais os produtos e regiões que estavam a drenar a margem de lucro da operação.

build A Abordagem Técnica

Utilizei Python para todo o processo de ETL (Extração, Transformação e Carga) e análise:

  • Limpeza: Tratamento de valores nulos e duplicados usando a biblioteca Pandas.
  • Exploração: Análise estatística descritiva (média, mediana, desvio padrão).
  • Visualização: Criação de gráficos de dispersão e barras com Matplotlib e Seaborn para identificar outliers.

Snippet de Análise em Python:

# Agrupando vendas por Região e calculando Lucro total
lucro_por_regiao = df.groupby('Region')['Profit'].sum().sort_values(ascending=False)

# Visualizando o resultado com Seaborn
sns.barplot(x=lucro_por_regiao.index, y=lucro_por_regiao.values)

lightbulb Insights & Resultados

A análise revelou que, embora a categoria "Móveis" gerasse uma alta receita bruta, os descontos excessivos aplicados para fechar essas vendas estavam a gerar um prejuízo operacional silencioso em 3 estados específicos.

Recomendação Estratégica: Revisar urgentemente a política de descontos para a categoria de Móveis e redirecionar o orçamento de marketing para a categoria "Tecnologia", que apresentou a maior margem de contribuição real.

database Visão de Extração via SQL

Para garantir que tenho a capacidade de extrair estes dados diretamente do banco de dados antes de os processar em Python, utilizaria uma query de agregação semelhante a esta:

SELECT
    Product_Category,
    SUM(Sales_Amount) AS Total_Revenue,
    COUNT(Order_ID) AS Total_Orders
FROM sales_table
WHERE Order_Date >= '2024-01-01'
GROUP BY Product_Category
ORDER BY Total_Revenue DESC;
