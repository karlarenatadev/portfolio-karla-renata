## 📖 Sobre o Projeto

Portfólio profissional interativo desenvolvido para apresentar soluções de **análise de dados**, **machine learning**, **automação RPA** e **desenvolvimento web**. Este ecossistema foi construído com foco em **performance**, **acessibilidade** e **narrativa de dados**, demonstrando como a tecnologia pode transformar números em decisões estratégicas.

### 🎯 Objetivos

- Centralizar projetos de dados com impacto mensurável em negócios.
- Demonstrar proficiência em automação de processos ponta a ponta.
- Proporcionar uma experiência de usuário moderna com design responsivo.
- Aplicar as melhores práticas de SEO e acessibilidade (WCAG).
