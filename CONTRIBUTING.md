# 🤝 Como Contribuir

Obrigado por demonstrar interesse em contribuir com este repositório! Seja reportando um bug, sugerindo uma melhoria de layout, ou ajudando na otimização de algum script de dados, sua colaboração é bem-vinda.

## 🛠️ Passo a passo para contribuir

1.  Faça um **Fork** deste repositório.
2.  Crie uma branch com a sua feature:
    * `git checkout -b feature/minha-nova-feature`
3.  Faça o commit das suas alterações utilizando *Conventional Commits*:
    * `git commit -m 'feat: adiciona novo dashboard de Power BI na galeria'`
4.  Faça o push para a sua branch:
    * `git push origin feature/minha-nova-feature`
5.  Abra um **Pull Request (PR)** detalhando o que foi alterado.

## 📝 Padrões de Commit (Conventional Commits)

Por favor, siga este padrão para manter o histórico claro:
* `feat:` Uma nova funcionalidade ou projeto.
* `fix:` Correção de um bug (ex: erro de responsividade).
* `docs:` Alterações apenas na documentação (README, arquitetura).
* `style:` Alterações de formatação (CSS, espaçamento) que não afetam a lógica.
* `refactor:` Uma mudança de código que nem corrige um bug nem adiciona uma funcionalidade.

## 🐞 Reportando Bugs

Se encontrar algum erro ou link quebrado, sinta-se à vontade para abrir uma **Issue**. Por favor, inclua:
* Descrição clara do problema.
* Passos para reproduzir o erro.
* Navegador e sistema operacional utilizados.

Agradeço por ajudar a manter este portfólio cada vez mais profissional!
